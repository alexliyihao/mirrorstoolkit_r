# Create a non-white color palette for ggplot2 based on Polychrome::glasbey.colors

Generate a color palette for ggplot2 with specified number of color,
white is intentionally removed.

## Usage

``` r
create_multi_color_palette(num_color)
```

## Arguments

- num_color, :

  int, the number of color need

## Value

list of str, the color list

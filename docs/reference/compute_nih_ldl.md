# NIH 2 equation, also known as the Sampson-NIH equation

Compute the LDL(calc) based on NIH equation 2

## Usage

``` r
compute_nih_ldl(tchol, hdl, trig)
```

## Arguments

- tchol, :

  float, total cholesterol

- hdl, :

  float HDL level

- trig, :

  float Triglyceride level

## Value

float, the LDL level
